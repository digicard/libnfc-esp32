# ESP32 port of libnfc

ESP32 port of [nfc-tools/libnfc](https://github.com/nfc-tools/libnfc) focused on PN532 and converted to a PlatformIO library.

- Ported to ESP32 platform (based on [this project](https://github.com/digicard/nfc_attendance_system_esp32/tree/main))
- CPP compiler compatibility
- Focused on PN532 NFC chip (all other chips support code deleted)
